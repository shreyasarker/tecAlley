# TecAlley
E-Commerce using Next, Strapi, Render, TailwindCSS and Stripe


### Project Link: https://tecalley.vercel.app

## Tech Stack
Frontend: Next.js
API: Strapi
Database: Render with PostgreSQL

## Frontend Support
- Next 13
- React 18
- React Dom
- React Toastify
- React Redux
- Reduxjs Toolkit
- Redux
- React Markdown
- React Multi Carousel
- React Responsive Carousel
- Stripe JS
- React Stripe JS
- Tailwind
- Auto PreFixer
- PostCSS

## API Support
- Strapi
- Cloudinary for Strapi
- Stripe
